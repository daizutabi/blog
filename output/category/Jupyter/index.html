JupyterLabのカスタマイズ

この記事では，JupyterLabをカスタマイズする方法を説明します．

1 JupyterLab

JupyterLabを実行する仮想環境を作成し，必要なパッケージをインストールします．

(base) conda create -y -n jupyter python=3.6 jupyterlab nodejs

2 jupyterlab-vim

VIMを使えるようにします．

(base) > activate jupyter
(jupyter) > jupyter labextension install jupyterlab_vim

1 Jupyter notebook extensions

base環境でインストールします．

(base) > conda install -c conda-forge jupyter_contrib_nbextensions

2 jupyter-vim-binding

(base) > jupyter --data-dir

で表示されるディレクトリの下に，nbextensionsディレクトリを作ります．そこに移動した後，
