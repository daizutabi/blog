Pheasantパッケージを作成しました

1. Fenced codeの実行

Markdownのfenced codeを自動で実行します．

Markdownのソースコードで，

```python
print(1)
```

と書くと，出力される結果は，

>>> print(2)
2

となります．

2. 図の出力

Matplotlibの図を出力することも可能です：

```python
%matplotlib inline
import matplotlib.pyplot as plt
plt.plot([1, 3]);
```

とすると，

>>> %matplotlib inline
>>> import matplotlib.pyplot as plt
>>> plt.plot([1, 3]);
